1. Игра в консоли на Python

В начале была написана игра, в которую можно было играть только в консоли, это заняло очень много времени и усилий, но в дальнейшем это очень помогло при переносе всего этого в веб-приложение. Хоть в итоге понадобились только фрагменты этого кода и некоторые идеи.

Очень хотелось сделать из этого что-то более красивое, я решил использовать для этого TKinter. Я уже сделал некоторую часть, когда понял, что в итоге получится не так замечательно как планировалось. Эта идея сошла на нет.

3. Использование Flask для создания веб-приложения

Flask — фреймворк для создания веб-приложений на языке программирования Python, использующий набор инструментов Werkzeug, а также шаблонизатор Jinja2.

4. Flask-SQLAlchemy для создания и работы с базой данных

Flask-SQLAlchemy — это расширение для фреймворка Flask, добавляющее поддержку SQLAlchemy в приложение. Модуль расширения направлен на упрощение использования пакет SQLAlchemy с Flask, предоставляя полезные значения по умолчанию и дополнительные помощники, облегчающие выполнение общих задач.

База данных состоит из двух таблиц: игр и игроков.

5. Bootstrap для более удобного оформления самой игры и других элементов приложения

Bootstrap — свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.
